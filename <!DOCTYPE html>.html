O que é HTML?

HTML(Hyper-Text-Mackup-Language) é o bloco de construção mais básico da web que define o significado e a estrutura do conteúdo da web

Hypertext

Hypertext refere-se ao que conectampaginas de web entre si, já dentro de um site ou entre sites

Marcação

A marcação são os chamadostagse elas comandam tudo. Os documentos HTML podem ser interpretadas por navegações
